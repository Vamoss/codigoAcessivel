Protótipo de editor de texto compatível com leitores de telas

O objetivo é proporcionar uma primeira experiência de programação para pessoas cegas utilizando o celular.

Abaixo você vai encontrar dois botões, um para editar o código e outro para executar o código.

No editor de código, cada linha é ativada por um botão. Após ativar a linha, você poderá editar cada bloco da linha separadamente.

Nesta fase do protótipo, é interessante saber se quebrar a linha em blocos ajuda ou dificulta a experiência de edição do código.

Como limitação conhecida, o editor ainda não trata erros e não permite quebra de linhas.
