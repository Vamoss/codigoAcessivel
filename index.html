Protótipo de editor de texto compatível com leitores de telas

O objetivo é proporcionar uma primeira experiência de programação para pessoas cegas utilizando o celular.

Abaixo você vai encontrar três botões: editar o código, executar o código e configurações de navegação e edição.

Em seguida você encontrará o código navegável por linha. É necessário ativar a linha para poder editá-la.

Como limitação conhecida, o editor ainda não permite quebra de linhas.
